## 🏆 Featured Project

### 🌍 [AidChain — Blockchain & Biometric Aid Distribution](https://github.com/motisha-24/blockchain-aid-distribution)

> *My dissertation capstone — solving humanitarian aid fraud in Zimbabwe*

A full-stack system that integrates **fingerprint biometrics**, **Ethereum smart contracts**, and **IoT hardware** to make aid distribution transparent, tamper-proof, and fraud-resistant.

**What I built:**
- ⛓️ **2 Solidity smart contracts** deployed on Ethereum Sepolia with duplicate-prevention logic
- 🔧 **1667-line ESP32 firmware** integrating fingerprint sensor + GSM SMS module  
- 🐍 **Flask REST API** with JWT auth, Web3.py blockchain bridge, offline cache
- ⚛️ **React dashboard** with 4 role-based views (Admin / NGO / Donor / Auditor)
- 📱 **React Native mobile app** with offline-first SQLite sync engine

```
Tech: Solidity · ESP32 · Flask · React · React Native · Web3.py · JWT · SQLite
```
